Troubleshooting: Missing Authorization Header

If you experience "Authentication required" errors on live environments despite sending a valid token, the Authorization header might be stripped by Apache.

Fix: Add this to your .htaccess: SetEnvIf Authorization "(.*)" HTTP_AUTHORIZATION=$1
